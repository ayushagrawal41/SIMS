# Student Information Management System(SIMS)
A Web Application which provides a simple interface for maintenance of student Information. It can be used by educational institutes or colleges to maintain the records of students easily.
# Tech Stack Used
HTML, CSS, JavaScript, MongoDB, ExpressJS, AngularJS and Node.JS.
# System Design
Login System

Student Details- Provides CRUD operations to student details.

Course Details- Provides CRUD operations to student details.

Attendance.

Marks and Grades.

Fee Details.
# Installation 
It requires Node.js v10+ to run.
Install the dependencies and devDependencies and start the server.

$ cd student-management-system

$ npm install -d

$ nodemon

AND

Import the users.json file to the local MongoDB server.

$ mongoimport --db student-mgmt-sys --collection users --file users.json

And run the application in the development mode.
